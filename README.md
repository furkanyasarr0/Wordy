**Wordy** is a lightweight, modern, and fast desktop application built with Python and Tkinter. It is designed to help you easily save, categorize, and manage words, phrases, or snippets across completely isolated workspaces. 

Whether you are learning a new language, saving coding snippets, or organizing your daily work, Wordy keeps your data safe and highly accessible.

## ✨ Features

* 📂 **Isolated Workspaces:** Create different environments (e.g., "Work", "English Vocab", "Coding") to keep your words neatly separated.
* 🏷️ **Multi-Tag Support:** Assign multiple tags to a single entry simply by separating them with commas (e.g., `python, database, urgent`).
* 🔍 **Live Search:** Instant, lightning-fast filtering as you type—no need to hit Enter!
* 🌙 **Dark & Light Mode:** Beautiful, modern UI that adapts to your preferences. (Saves your choice automatically).
* 💾 **Auto-Backup System:** Never lose your data. Wordy can silently back up your workspace to a `.json` file after every change.
* 🔄 **JSON Import / Export:** Easily migrate your data, create manual backups, or share your word lists with others.
* 📌 **Always on Top:** Pin the application to stay above all other windows for quick copy-pasting while working.
* 🚀 **High Performance:** Built-in pagination (50 items per page) ensures the app runs smoothly even with thousands of words.
* 🛡️ **Duplicate Prevention:** Automatically warns you if you try to add the same word in the same workspace.

## 🚀 Installation & Usage

Wordy is built entirely using Python's standard library. **No external dependencies** (like `pip install ...`) are required!

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/furkanyasarr0/Wordy.git](https://github.com/furkanyasarr0/Wordy.git)
    cd Wordy
    ```

2.  **Run the application:**
    ```bash
    python app.py
    ```

*(Note: The database `wordy.db` will be automatically generated in the same folder on your first run.)*
